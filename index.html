网格参数化算法实现

课程作业

基于 OpenMesh 实现多种网格参数化算法，包括 Tutte、ARAP、ASAP 及 Hybrid 方法。使用 Eigen 求解稀疏线性系统，支持 cotangent 权重和均匀权重的灵活切换。ARAP 算法通过双坐标拉普拉斯矩阵构建，实现高效的局部旋转与全局松弛两阶段优化。

C++ OpenMesh Eigen 计算机图形学
◉

光栅化与路径追踪渲染器

课程作业

基于 Ruzino 框架构建完整渲染管线，实现 Blinn-Phong 光照、Shadow Mapping、PCSS 软阴影与 SSAO 环境遮蔽。路径追踪模块采用蒙特卡洛积分采样，配合俄罗斯轮盘赌技巧实现无偏光照估计，支持多种材质与复杂场景的真实感绘制。

C++ OpenGL 路径追踪 物理渲染
∫

龙贝格积分算法

课程作业

实现高精度 Romberg 积分算法，支持奇异点处理与无穷区间积分。采用 Richardson 外推加速收敛，配合自适应区间细分策略。算法可处理端点奇异性、对数奇异性等复杂情况，适用于物理与工程中的定积分计算。

C 数值分析 科学计算 数值积分
⚗

飞行时间质谱研究

大学生研究计划 项目编号：20241172 获评等级：A

指导教师：赵龙、许鸣皋
研究时间：2024年12月—2025年12月

研究内容：围绕反射式飞行时间质谱（Reflectron TOF-MS）的核心物理机制，针对单反射式与双反射式两种典型结构，深入推导了实现最高分辨率条件下所需理想电场的精确数学表达式。系统推导了能量聚焦所需的均匀静电场条件，以及双反射式结构中不同电场区域的场强分布与几何尺寸的耦合方程。

使用工具：Maple（符号计算推导公式）、SIMION（电场模拟）
论文成果：《基于同步辐射的高分辨高灵敏反射式飞行时间质谱研制》（待发表，3/6）

科学研究 质谱系统 Maple SIMION
